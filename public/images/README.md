# Menu Images

This directory contains SVG recreations of authentic restaurant menu pricing from Vancouver, BC's Pho Goodness restaurant.

## Files

### menu-2015.svg
- **Source**: [Wayback Machine Archive](https://web.archive.org/web/20150309061033/http://www.phogoodness.com/menu.html)
- **Archive Date**: March 9, 2015
- **Key Pricing**: Pho Dac Biet (Special) at $8.50, other pho varieties at $8.00
- **Design**: Recreated from original menu board with vintage Vietnamese restaurant aesthetic
- **Purpose**: Demonstrates baseline restaurant pricing before the Vancouver restaurant industry squeeze

### menu-2025.svg
- **Source**: Current Pho Goodness menu (accessed November 2, 2025)
- **Key Pricing**: Pho Dac Biet at $17.00, other pho varieties at $16.00
- **Design**: Modern clean web menu design reflecting current restaurant presentation
- **Purpose**: Shows current pricing to illustrate 100% price increase over 10 years

### menu-backdrop.svg
- Background graphic element for menu presentation scenes

## Data Integrity

These SVG files are based on real menu data sourced from:
1. Wayback Machine historical archives (2015)
2. Current restaurant website (2025)

The pricing information is authentic and verifiable. See `/public/data/sources.json` for complete documentation including:
- Archive URLs
- Exact pricing comparisons
- Context about inflation vs industry-specific cost pressures

## Usage in Application

These menu images are used in the scrollytelling narrative to visually demonstrate the dramatic price increases in Vancouver's restaurant industry, specifically highlighting that a 100% price increase far exceeds general inflation (~27% over the same period).
